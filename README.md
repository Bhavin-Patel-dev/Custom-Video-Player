# 🎥 Custom HTML5 Video Player

A small practice project built as part of **project-based learning** to better understand how the HTML5 Video API works using **Vanilla JavaScript**.

The goal of this project was to build a fully functional custom video player from scratch without relying on external libraries or frameworks, while exploring DOM manipulation, event handling, and media controls.

---

## 📌 Purpose

This project was created to:

- Learn the HTML5 Video API
- Practice JavaScript DOM manipulation
- Understand event-driven programming
- Build custom media controls instead of using the browser's default player
- Strengthen problem-solving through hands-on project-based learning

---

## 🚀 Features

- ▶️ Play & Pause video
- ⏩ Skip forward and backward
- 🔊 Volume control
- ⚡ Playback speed adjustment
- 📊 Live progress bar updates
- 🎯 Seek video by clicking the progress bar
- 🎨 Custom responsive video player UI

---

## 📚 Learning Outcomes

While building this project, I learned about:

- HTML5 `<video>` element and Media API
- DOM querying and manipulation
- Event listeners and event handling
- Working with media properties like:
  - `currentTime`
  - `duration`
  - `paused`
  - `volume`
  - `playbackRate`
- Updating UI dynamically based on application state
- Creating reusable JavaScript functions
- Building interactive user interfaces using only HTML, CSS, and JavaScript


No installation or dependencies are required.
